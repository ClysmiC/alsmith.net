Hi, I am Andrew L. Smith - Programmer

In a hurry? Just read the bold and the bullet points.

I'm a 22 year-old college student from St. Louis, Missouri, studying computer science at Georgia Institute of Technology. I started programming when I was 14, learning how to make simple 2D games with a program called Game Maker. I started taking CS classes at 16 and have been trying to learn more ever since. Most of my work experience is software development, but I have been exposed to a lot more than that in my class and personal projects-- check them out. Or look at my github! I only started using github a few years ago, so it doesn't have all my projects but I am trying to add to it as I work on new things!

Here is a quick run-down of what languages I have worked frequently with:

  • Java (+ Android Programming)
  • C#
  • Python (+ Tornado)
  • LC-3 assembly
  • GML
  • Basic HTML/CSS (enough to make this site!)

And a large assortment of languages that I have use to varying degrees in classes (but haven't used too frequently or recently). The higher on this list, the more comfortable I am with it:

  • C (+ Arduino)
  • C++ (+ mbed)
  • Processing
  • Javascript + JQuery
  • GLSL
  • SQL
  • ARM assembly
  • VHDL

For a better picture of my skill-set, check out my resume.

Lastly, here are some random things about me that may or may not be of interest:

  • I played baseball for 11 years. After that, I umpired for 5.
  • I've been using the Dvorak keyboard since 2007.
  • I can eat six saltines in a minute.
  • I love video games. They are a big part of what got me interested in computers (and subsequently, programming)--I'd love to work on one professionally one day.
  • I'm currently undergoing the process of learning Emacs. This site is the first thing I've written using it.
  • I'm a die-hard St. Louis Cardinals fan.
  • When given a choice, I use Allman style brackets.
  • I do a crossword puzzle almost every day.
  • I can beat an intermediate board of minesweeper in under 30 seconds.
